N3w world server - About

1 版本

1.18.2,Java17及以上

低端机可以尝试使用OpenJ9虚拟机游玩可以节省内存占用。

1.1 OpenJ9的配置

第一步:检查你的系统版本

不同的系统会有不同的文件后缀名且不同系统之间不互通,例如Windows上的文件(.exe)不能与macOS上的文件(.app)互通

*Linux系统一般都会有一些分支:例如Ubentu这些分支其实本质上都是Linux系统,只是修改了一些东西而已

*OpenJ9的版本都有Windows,Linux,MacOS,z/OS,AIX


第二步:检查你的系统是否64位

不同的系统架构会有不一样的运行结果,64位可以运行32位程序但不能运行16位程序,32位可以运行16位程序但不能运行64位程序

OpenJ9的Windows只有64位版本,没有ARM64

*OpenJ9其他系统版本不只支持64位版本!Linux可以选择ppc64le,s390x和aarch64,Mac可以选择aarch64,AIX可以选择ppc64

如果你下载了一个不支持你的系统架构的版本,那么你无法运行它。


第三步:选择版本

在表格中会有系统、架构、版本信息。建议使用最新版的Java.


第四步:下载

直接使用浏览器下载是最不明智的行为(除非你的wifi很快)

你可以考虑使用IDM下载器下载它,IDM的正版需要购买验证码,否则只有30天使用时间

你可以使用破解版来下载IBM网站上的文件即可满速下载


第五步:解压

如果你直接使用浏览器下载,那么在你没设置的时候一般放在"下载(downloads)"文件夹内(Windows下),那么请你打开"文件资源管理器(explorer.exe)"后找到"下载"文件夹

如果你使用IDM下载,那么它的压缩包将在Compressed出现(例如JDK18是ibm-semeru-open-jdk_x64_windows_18.0.1.1_2_openj9-0.32.0.zip)

使用Bandizip等程序解压它到你所指定的文件夹内(要记住,有用!!)(这里假设你放在 disk1/java/OpenJDK18内)


第六步:选择一个启动器

这里推荐使用PCL2启动器来启动该游戏,该游戏在此网页上有可供下载的Fabric Minecraft副本,直接放到对应的文件夹下方即可。

如果你不想使用PCL2,你可以考虑试一试HMCL,它同样可以使用网页上的整合版本。


第七步:将整合包放入文件夹中

在本网站下载整合包然后放入/.minecraft/version文件夹内即可。


第八步:最后设置

第一:在启动器内开启 版本隔离选项 如果是PCL2,请选择隔离所有版本。其他开启即可。

第二:运行游戏


第九步:开启!


2 建议装载的模组

本服务器对离线玩家是开放的,你无需花费89￥(6月12日更新)购买Java和Bedrock版本。这样节省了不必要的花费。

但是离线的服务器往往不能显示服务器内玩家的皮肤,这真是个大麻烦...不过我们可以使用mod解决这个问题。

我这里介意4GB的PC使用fabric,8GB的使用forge.因为forge非常消耗内存。Java的原理是虚拟机,所以运行效率会很差。

建议使用64位的PC并使用64位的Java版本,目前OpenJ9已经弃用32位版本。

待一切准备完毕之后,你可以安装模组了。那么下载CustomSkinLoader是显示皮肤的最佳选择,它可以看到所以玩家的皮肤,不管他多么冷门。

如果你是一个fabric用户并且PC未达到推荐配置(Java版本),建议去掉Opifine和Opifabric.这两个与优化模组不兼容。

你可以使用钠模组来优化你的Minecraft加载速度来提升帧率,即便你舍弃了Opifine光影。

3 开启游戏

建议你使用PCL2打开游戏(详见上文),目前PCL2是我的世界启动器的最佳选择。或者你也可以试一下HMCL。如果你想体验官方启动的话可以从minecraft.net下载一个我的世界启动器。

过程很简单...只需要按下"开启游戏"按钮即可开启了!

4 服务器制度

没有规矩,不成方圆。这句话在哪里都是适用的,在你游玩的过程中,为了能让其他玩家也可以获得一个良好的游玩环境。特列出服务器规定如下:(未决定)

4.1 N3w World Minecraft Server 游玩规定(未决定版)

1.不得破坏其他玩家的建筑(判决生效条件:30方块以上 且物品损失估价在15绿宝石以上)

2.不得偷窃玩家重要物品(判决生效条件:估价在13绿宝石以上)

3.禁止制造卡服机、禁人塔、禁人书等违规物品和建筑(判决生效条件:发现即生效)

4.禁止DDoS等对服务器的网络攻击(判决生效条件:发现即生效)

5.服务器配置不高,请尽可能不要高强度跑图。(当然必须得跑得话也是可以的)

6.服务器内玩家尽可能会帮助资源不太好的玩家,但是玩家最好是自己获得资源并使用

7.服务器内"失物招领处"等建筑禁止盗窃,盗窃者将会被重罚(判决生效条件:发现即生效 将会以同样估价的物品赔偿,若没有则直接强制处理)

8.服务器玩家多数因学业原因不能上线进行游玩,请谅解。

9.服务器玩家最好控制在10个左右的人即可,过多将会导致服务器卡顿。

10.禁止开挂,本服务器对开挂玩家零容忍。(判决生效条件:发现即ban)

11.服务器允许使用矿物透明等寻找矿物。

12.请文明发言。(判决生效条件:发现即生效)

13.除服务器风评不好外请不要诋毁服务器导致服务器风评被害。(判决生效条件:发现即生效)

14.服务器因为某些原因没有关闭禁止飞行,在高ping情况下会被踢出游戏

15.服务器不会让任何外人当上管理员,请不要在聊天栏内发送求管理员权限的消息,服务器管理员(一般是老玩家)会对此事件进行处理。

16.服务器不是网易某些xxs的藏身之地,禁止ghs或者搭建违规建筑,发现会被拆掉。(判决生效条件:发现即生效)

17.请不要在village,village4,village5,village6进行袭击,这些村子尚未准备好应对袭击。

18.在游玩过程中,某些玩家可能会因为某些原因大战一场,请不要对玩家进行劝架、教唆打架等行为。

19.在游玩之前,你应当熟悉minecraft原版操作再进入游戏。关于键位问题不会进行回答(部分玩家会)。

20.游玩期间请不要发送引战内容:例如迷你世界等言论导致言论战爆发(发现即生效)

4.2 N3w World Minecraft Server 处罚规定(未决定版)

[危险等级 0] - 睁一只眼闭一只眼就过去了

[危险等级 1] - 会被提示或劝告

[危险等级 2] - 会被警告不要再这样做

[危险等级 3] - 会被管理员/服主警告

[危险等级 4] - 会被踢出

[危险等级 5] - 有期限的封禁(10天以下)

[危险等级 6] - 有期限的封禁(10~40天)

[危险等级 7] - 无期限永久封禁

[危险等级 8] - ban-ip

[危险等级 9] - 介入调查

[危险等级 10] - (针对服务器的DDoS等攻击且造成了规定的损失)走法律程序(已经造成了规定的损失)

5.游玩技巧

在你死亡之后,你可以输入/back回到死亡的位置

如果你要自杀回出生点,请输入/suicide回到出生点(再见了,这残酷的世界.jpg)

出生点一定范围之内放置、修改、挖掘任何东西无效

服务器可以存储一个玩家的5个home传送点,使用/sethome设置一个家,使用/home (家的昵称)回到家,使用/delhome删除一个家(普通玩家不能跨世界传送)

服务器可以使用warp地标传送,可以跨世界传送。这也是指令标红错误解决的唯一方式

指令错误标红可以输入/warp exp来传送到末地小黑经验塔再次加载世界即可恢复正常,然后再回到主世界传送点即可(/home不能跨世界,/warp可以)

建议安装小地图来查看玩家的位置

在苦手的时候,请叫来其他的空闲玩家帮忙。
